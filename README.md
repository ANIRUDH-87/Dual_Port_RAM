## 📌 Project Overview  
This project implements a **Dual-Port RAM (DPRAM) using SystemVerilog**, allowing **simultaneous read and write operations** on two independent ports. Unlike **Single-Port RAM**, which only allows one operation (read or write) at a time, Dual-Port RAM enhances **parallel processing** by enabling two processes to access memory concurrently. This design is particularly useful in **high-speed computing, FPGA-based architectures, and embedded systems**.  

✅ **Supports simultaneous read/write on two ports**  
✅ **Parameterized design for flexible memory depth and width**  
✅ **Implements conflict resolution for write collisions**  
✅ **Synchronous and asynchronous read support**  
✅ **Fully verified testbench in SystemVerilog**
